In order to identify playing music you need to sign up for an API key with audd.io

They offer 300 free requests, which should be enough to get you up and running and to test out the system

To get an API key you need to use their Telegram bot but please note the following important warning

Important note about audd.io

audd.io is not a free service.

They offer a trial period for 2 weeks which gives you 300 requests, which you can request via the Telegram bot.

Once you have exhausted this you will need to sign up for one of their paid plans. These can be fairly affordable, and at the time of writing they offer 1000 requests per month for $2.

However please note that if you sign up for a plan and send a lot of requests them you can run up considerable bills. For instance 100,000 requests will cost $420. Once you sign up for a plan they will let you carry on sending API requests for as long as you want and then send you the bill.

Be careful how many API requests you send! You can monitor your usage via their Telegram bot.
